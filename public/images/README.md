# 画像の差し替え方法

このフォルダのファイルを同名で上書きするだけで、サイト全体の画像が差し替わります。

| ファイル名 | 使用場所 | 現在の内容 |
| --- | --- | --- |
| `logo.jpg` | （元データ）ロゴ全体 | 染マーク＋YUI COLORSロゴ |
| `logo-mark.jpg` | ヘッダー・フッターのロゴ | 丸い染マーク（logo.jpgから切り出し） |
| `og-image.jpg` | SNSシェア時のOGP画像（1200×630） | ロゴ入りOGP（自動生成） |
| `hero.jpg` | トップのファーストビュー背景 | 琵琶湖を望む窓辺での体験風景 |
| `dyeing-01.jpg` | トップ「とは」・ストール体験・雨の日導線 | グラデーションのストール |
| `workshop-01.jpg` | ハンカチ染め体験・ブログ | 絞りの下準備をした白い布 |
| `workshop-02.jpg` | トートバッグ染め体験・ブログ | 水色の絞り染めトート |
| `kids.jpg` | 親子向け導線・親子ページ | 桜柄の巾着袋 |
| `date.jpg` | デート向け導線・デートページ | ふたり並んでの作業風景 |
| `lake.jpg` | 観光導線・観光ページ・ブログ | 琵琶湖の夕景 |
| `access.jpg` | アクセスページ | 店舗入り口の外観 |
| `gallery-01.jpg`〜`gallery-08.jpg` | トップのギャラリー | 体験・作品・工房の写真8枚 |

## 注意

- ファイル名は上記と完全に一致させてください（小文字・ハイフン）。
- JPG形式・1枚あたり500KB以下を目安に圧縮すると表示が速くなります。
- ギャラリーの写真を増やす場合は、ここにファイルを追加して `data/gallery.ts` に追記してください。
- 元の写真データ（LINEアルバム一式）はプロジェクト直下の `photos-original/` に保管しています。
