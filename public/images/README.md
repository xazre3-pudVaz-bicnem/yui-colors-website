# 画像の差し替え方法

このフォルダに下記のファイル名で写真を置くだけで、サイト全体の画像が差し替わります。
写真が置かれるまでは、自動的に上品なプレースホルダーが表示されます。

| ファイル名 | 使用場所 | 推奨サイズ（目安） |
| --- | --- | --- |
| `hero.jpg` | トップページのファーストビュー背景 | 2000×1250px 程度（横長） |
| `dyeing-01.jpg` | 染めの様子（トップ「とは」・体験ページ・ギャラリー） | 1200×1500px 程度（縦長可） |
| `workshop-01.jpg` | 体験メニュー1（ハンカチ染め）・ギャラリー | 1200×800px |
| `workshop-02.jpg` | 体験メニュー2（トートバッグ染め）・ギャラリー | 1200×800px |
| `kids.jpg` | 親子向け導線・親子ページ・ギャラリー | 1600×900px |
| `date.jpg` | デート向け導線・デートページ・ギャラリー | 1600×900px |
| `lake.jpg` | 観光導線・観光ページ・ギャラリー | 1600×900px |
| `access.jpg` | アクセスページのエリア写真 | 2000×860px 程度（横長） |
| `og-image.jpg` | SNSシェア時のOGP画像 | 1200×630px（固定） |

## 注意

- ファイル名は上記と完全に一致させてください（小文字・ハイフン）。
- JPG形式・1枚あたり500KB以下を目安に圧縮すると表示が速くなります。
- ギャラリーの写真を増やしたい場合は `data/gallery.ts` に追記してください。
